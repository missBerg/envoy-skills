# Envoy Gateway Skills

[Envoy Gateway](https://gateway.envoyproxy.io/) is a Kubernetes-native API gateway that provides a simplified interface for managing Envoy Proxy as an edge, API, or Kubernetes ingress gateway. It implements the Kubernetes Gateway API and extends it with Envoy-specific features.

## Audiences

### [Adopters](adopters/)

For developers deploying and configuring Envoy Gateway in their infrastructure. Covers installation, routing, security, TLS, rate limiting, extensions, observability, and production hardening.

Start with the `eg-orchestrator` agent which will interview you about your use case and guide you to the right skills.

### [Contributors](contributors/)

For developers contributing code to the [envoyproxy/gateway](https://github.com/envoyproxy/gateway) repository. Covers codebase navigation, coding standards, testing, and PR workflows. *(Coming soon)*

## Compatibility

These skills target:
- Envoy Gateway v1.3.x (latest stable)
- Kubernetes Gateway API v1.2
- Kubernetes 1.28+
